HOW TO PLAY

Grow your own randomly generated flower by matching the tiles. Click and drag the tiles to move them into the slots at the top. You can generate a new flower and start the game over at any time.

Click here to download an EXE installer for the game.

HOW TO MINT YOUR FLOWER

Once your flower is fully grown you can mint it to the Tezos blockchain. Clicking mint in game will open a browser page. The interface is not hosted publically yet, so you will need to start your own local server according to the Backend and Frontend Usage instructions found here.

This game is not yet available on the Tezos mainnet. Minting is on hangzhounet (test net), meaning the NFT does not have any value. The wallet you mint to must be a hangzhounet wallet.

For minting to work a secret.json file must be added to the Create-Tezos-NFT folder with Pinata API keys as described here.
