# Little-Flower
Little Flower Video Game for PC written in C++.
See the [Game Design Document](https://www.notion.so/reganwillis/Little-Flower-Game-Design-Document-332e2eabf06e401bb381adc6e64a6043).

## Development
I used [this tutorial](https://code.visualstudio.com/docs/languages/cpp#_example-install-mingwx64) and [this tutorial](https://www.msys2.org/) to install a C++ compiler and add it to the path. Compile the program with `g++ main.cpp` and run with `a.exe`.
